# Escape From Tiamat
A browser-based horror-escape game built with Three.js.  
Developed for Computer Graphics course.  




Requirements:   
Node.js (v18 or newer recommended)  




Setup:
```bash
  
npm install       #install dependencies, only for the first time  
npm run dev       #start the build  
```
(Then open the URL shown in the terminal (usually http://localhost:5173) in your browser.)  


Notes:  
The game requires pointer lock. Click on the screen to activate mouse control, press Escape to release it.
